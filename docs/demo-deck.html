04 / Feature Workspace
The Core Idea

Every role. Same context.
Zero re-briefing.

07 / Tech
Technical Approach

Production-ready stack.
Runs in Docker.

🤖 Agent Layer

LangGraph — route → retrieve → reason → act. Live Jira/Confluence API calls on every query. Confirm-before-write for all mutations.

🧠 Intelligence

Claude Sonnet for reasoning & drafting. Claude Haiku for classification & extraction. all-mpnet-base-v2 for local embeddings.

🗄️ Knowledge Store

PostgreSQL + pgvector. Memories never auto-deleted. Feature workspace persists sessions, takeaways, and artefact links.

⚡ Infrastructure

FastAPI + SSE streaming. Celery for async sync. Next.js UI. Docker Compose — one command to run everything.
